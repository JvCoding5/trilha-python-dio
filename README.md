# Trilha Python DIO

Este repositório contém uma coleção de exercícios e projetos desenvolvidos durante a trilha de aprendizado de Python oferecida pela Digital Innovation One (DIO). O objetivo é proporcionar uma base sólida em Python, abordando desde os fundamentos até conceitos avançados, como programação orientada a objetos e desenvolvimento de APIs.

## Estrutura do Repositório

O repositório está organizado em várias pastas, cada uma representando um tópico específico. Abaixo está uma descrição de cada uma:

### 00 - Fundamentos
Nesta pasta, você encontrará os conceitos básicos de Python, incluindo:
- Tipos de dados
- Estruturas condicionais
- Estruturas de repetição
- Operadores

### 01 - Estrutura de dados
Aqui, são abordadas as principais estruturas de dados em Python, como:
- Listas
- Tuplas
- Conjuntos
- Dicionários

### 02 - Programação Orientada a Objetos
Esta seção cobre os princípios da programação orientada a objetos, incluindo:
- Classes e objetos
- Herança
- Encapsulamento
- Polimorfismo

### 03 - Decoradores, Iteradores e Geradores
Explora conceitos avançados como:
- Decoradores
- Iteradores
- Geradores

### 04 - Data e hora
Trabalha com manipulação de datas e horas em Python.

### 05 - Manipulação de arquivos
Ensina como ler e escrever arquivos, além de boas práticas de manipulação de arquivos.

### 06 - Gerenciamento de pacotes e boas práticas
Discute o uso de gerenciadores de pacotes e boas práticas de programação.

### 07 - Banco de dados
Aborda a interação com bancos de dados usando Python.

### 11 - Desenvolvimento de APIs com Flask
Introduz o desenvolvimento de APIs utilizando o framework Flask.

### 12 - Desenvolvimento fullstack com Django
Cobre o desenvolvimento de aplicações web utilizando o framework Django.

### 13 - APIs Assíncronas com FastAPI
Explora o desenvolvimento de APIs assíncronas com FastAPI.

### 14 - Rascunhos
Esta pasta contém projetos e ideias em desenvolvimento.
